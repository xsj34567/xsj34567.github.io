## 3. 删除重复数据

### 3.1 删除重复数据并保留一条

```sql

DELETE 
FROM
	student 
WHERE
	id NOT IN 
	( 
    	SELECT id FROM 
    	( 
        	SELECT min( id ) AS id 
        	FROM student GROUP BY id_card, `name` 
    	) AS a 
	);

```


```sql
-- zmm

delete from student a 
where a.id != 
(
select min(b.id) from student b where a.name=b.name and a.id_card = b.id_card
) ;


```
